syntaxhighlighter使用与normalize.css阅读

写上一篇的时候尚不知道sh高亮，以致代码块极为原始。现在来用用syntaxhighlighter

看下面一段代码： 
            fieldset {
            padding: .35em .625em .75em;
            margin: 0 2px;
            border: 1px solid #c0c0c0;
            }
            legend {
            padding: 0;
            border: 0;
            }
这表示：表单分组标题的内添充为上 .23em，下 .75em， 左右 .625em。外边距为上下0，左右2px，边框为#c0c0c0、实心的1px宽。 分组标题的内添充为0，边框为0。

这样子代码看起来确实舒服很多啊。继续看看代码

            textarea {
            overflow: auto;
            }
             optgroup {
             font-weight: bold;
            }

多行文本框在需要滚动条的时候再显示,选项组的字体样式为加粗。

            table {
            border-spacing: 0;
            border-collapse: collapse;
            }
            td,
            th {
             padding: 0;
            }

表格table标签的相距单元的边框之间的距离为0，并且合并。单元格、标题行的单元格的内添充为0. 可以设置2个值，第1个为左右，第2个为上下。如果想继承自父容器则设为inherit border-collapse是说如何处理相临单元格之间的边框样式。这里设置的是合并。 有3个值可选。

          * {
             color: #000 !important;
            text-shadow: none !important;
            background: transparent !important;
            -webkit-box-shadow: none !important;
            box-shadow: none !important;
           }

全局设定，字体颜色黑，清除字体及盒标签的阴影，背景透明。 ！important，设置为最高优先级，覆盖同类型的样式设置。

           a,a:visited {
              text-decoration: underline;
            }
             a[href]:after {
                 content: " (" attr(href) ")";
            }
             abbr[title]:after {
             content: " (" attr(title) ")";
            }
             a[href^="javascript:"]:after,
             a[href^="#"]:after {
             content: "";
             }

1、链接的默认及访问过的样式为加上下划线。

2、链接的href属性内容后面加上属性href的值，举例子：

百度，实际的显示内容为 百度(www.baidu.com)

3、链接的title属性的内容后面加上属性title的值。

4、如果链接的href的值是以javascript和#开头的，那清除掉插入的任何值，以免调用js及跳转锚点出错。

5、缩写abbr的title属性的内容后面加上属性title的值。 content 属性与 :before 及 :after 伪元素配合使用，来插入生成内容。
