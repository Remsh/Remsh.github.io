用ajax完成表格的简单提交

提交表单的方法很多，用ajax，将input参数逐个传递到Data中，是很常用的办法，代码如下:



            $.ajax({
            type: "POST",
            url: "some.php",
            data: "name=John&location=Boston",
            success: function(msg){
            alert( "Data Saved: " + msg );
   }
});

        $.ajax({

            }).get().join("&") ;
        type: "POST",
        url: "some.php",
        data: str_data,

     alert( "Data Saved: " + msg );
   }
